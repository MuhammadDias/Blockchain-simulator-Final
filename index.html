Hash

Konsep dasar fingerprint data digital.

Hash adalah fungsi satu arah yang mengubah input data menjadi string karakter dengan panjang tetap. Di sini kita menggunakan SHA-256 untuk melihat bagaimana perubahan kecil pada data mengubah output secara drastis.

Block

Simulasi mining & Proof of Work.

Block adalah wadah data dalam blockchain. Anda akan belajar tentang Nonce dan tingkat kesulitan (Difficulty) dalam proses mining untuk menemukan hash yang valid (dimulai dengan 0000).

Blockchain

Rantai blok yang saling terikat.

Melihat bagaimana blok-blok terhubung satu sama lain menggunakan "Previous Hash". Jika satu blok diubah datanya, semua blok setelahnya akan menjadi tidak valid (rusak).

ECC Signature

Tanda tangan digital & Kunci privat.

Elliptic Curve Cryptography (ECC) digunakan untuk membuat pasangan kunci Public & Private. Ini memungkinkan penandatanganan transaksi yang aman dan verifikasi kepemilikan tanpa membocorkan kunci rahasia.

Konsensus

Kesepakatan data antar user.

Simulasi jaringan peer-to-peer terdistribusi di mana User A, B, dan C memiliki salinan database sendiri. Pelajari bagaimana mereka mencapai kesepakatan (konsensus) atas transaksi yang valid.
